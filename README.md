# pytest_ui_api_template

## ## Шаблон для автоматизации тестирования на python

### Шаги
1. Склонировать проект 'git clone https://github.com/romanlf64/pytest_ui_api_template.git'
2. Установить зависимости 'pip install -r requirements.txt'
3. Запустить тесты 'pytest', 'pytest -m api', 'pytest -m ui'
4. Сгенерировать отчет 'pytest --alluredir=./allure-result'
5. Открыть отчет 'allure serve allure-result'

### Стек
- selenium
- pytest
- requests
- allure

### Структура
- ./test - тесты
- ./pages - классы
